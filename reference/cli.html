    Cli Reference¶

    csml.py¶

    csml.py [OPTIONS] COMMAND [ARGS]...

    run¶

    Run pipeline defined in config

    csml.py run [OPTIONS] OUTPUT

    Options

    -c, --config <config_file>¶

    Required YAML config file

    -s, --slice <slice>¶

    id_slice column value

    Default:¶

    0

    --continue¶

    don’t clear the temporary database

    Default:¶

    False

    Arguments

    OUTPUT¶

    Required: output sqlite database

    check-scival-fields¶

    Check if CSV fields correspond to those defined in config file

    csml.py check-scival-fields [OPTIONS] GLOB HEADERLENGTH

    Options

    -c, --config <config>¶

    YAML config file

    Arguments

    GLOB¶

    Required: Glob to find CSVs

    HEADERLENGTH¶

    Required: number of lines to skip, or ‘auto’ for auto-detection

    config-schema¶

    Export jsonschema for config

    csml.py config-schema [OPTIONS]

    Options

    -o, --output <output>¶

    Output JSON file. If omitted, will print to stdout

    --docs¶

    Ensure compatibility with sphinx-immaterial

    Default:¶

    False
